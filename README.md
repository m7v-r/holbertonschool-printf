# 0x11. C - printf

## Description
This project is a custom implementation of the C standard library function `printf`. The function `_printf` writes output to `stdout` according to a specific format.

## Prototype
`int _printf(const char *format, ...);`

## Supported Conversion Specifiers
| Specifier | Description |
|-----------|-------------|
| `%c`      | Prints a single character |
| `%s`      | Prints a string of characters |
| `%%`      | Prints a percent sign |
| `%d`      | Prints a decimal (base 10) number |
| `%i`      | Prints an integer |
